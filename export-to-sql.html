WayPoint → Supabase Seed Generator

Loads the current hardcoded data files and produces ready-to-run INSERT statements that seed your Supabase universities and cities tables.

How to use

  1. First, run supabase-schema.sql in the Supabase SQL Editor (one time only — creates the tables and RLS policies).
  2. Click Generate SQL below.
  3. Click Copy to clipboard (or Download .sql).
  4. Open the Supabase SQL Editor → New Query → paste → Run.

The generated script is safe to re-run — every row uses ON CONFLICT (id) DO UPDATE, so re-importing just refreshes the existing rows in place.
